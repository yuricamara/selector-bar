Cada seletor colorido da barra representa um dos objetos contidos no array options exceto os seletores amarelos que são meros placeholders.

A barra é criada dinamicamente de acordo com o valor do array options.Os objetos deste array com propriedade group igual a red são colocados a esquerda da barra e recebem a cor vermelha. Os objetos com propriedade group igual a green são colocados após os seletores do grupo1 e recebem a cor verde.

Use CTRL para selecionar mais de um seletor. Os objetos selecionados são adicionados ou retirados de selectedObject na ordem.
